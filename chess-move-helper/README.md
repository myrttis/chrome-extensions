# ♟ Chess.com 走位助手 (Move Helper)

Chrome 浏览器扩展，在 chess.com 下棋时显示棋子合法走位 **并自动分析将杀路线**。

## ✨ 功能

### 走位标记
| 标记 | 含义 |
|------|------|
| 🟢 绿色小圆点 | 可移动到的空格 |
| 🟠 空心圆环 | 可吃子的位置 |
| 🟡 黄色底色 | 当前选中的棋子 |
| 🔵 蓝色圆点 | 王车易位 |
| 🟣 紫色圆点 | 过路兵 |

### 将杀路线分析
| 标记 | 含义 |
|------|------|
| 🔴 红色大圆点 | 一步将杀 (Mate in 1) |
| 🔴 红色底圆环 | 一步将杀吃子 |
| 🟠 橙色圆点 | 强制将杀 2步内 (Mate in 2) |
| 🟠 橙色半透明 | 强制将杀 3步内 (Mate in 3) |

选中有将杀路线的棋子时，右上角会弹出 **将杀路线面板**，显示完整走法序列。

### 快捷键
- **Ctrl+Shift+M** — 开关扩展

## 📦 安装方法

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `chess-move-helper` 文件夹
5. 打开 chess.com 即可使用

## 🔧 使用方式

1. 打开 chess.com 对弈页面
2. **点击棋子** → 显示所有合法走位 + 自动分析将杀路线
3. 如果发现将杀，棋盘上用红色/橙色标记将杀目标格，右上角显示将杀路径
4. 再次点击同一棋子取消选择

## 🧠 将杀分析引擎

- **Mate in 1**：遍历所有走法，检查是否直接将杀
- **Mate in 2**：走一步后，检查是否所有对手应对都导致我方下一步将杀
- **Mate in 3**：对剩余走法做 2 层深度的强制将杀搜索
- 使用 chess.js 引擎保证走法合法性
